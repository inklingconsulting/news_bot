What it is

A bot that reads the news so you don't have to — then sends you the five things worth knowing, and learns which ones you actually care about.

Every morning, Claude searches the web, writes a tight brief across the categories you choose, and drops it into your Telegram. Tap to rate each story; tomorrow's edition shifts toward your taste.

The loop

Research, deliver, learn

01 — Research

Claude reads the web

One API call with the web-search tool gathers the freshest real stories, judges relevance, and writes tight summaries with sources.

02 — Deliver

It lands on your phone

The brief arrives in Telegram at your chosen hour — each story its own message with 👍 / 👎 buttons.

03 — Learn

Your taste steers it

Every tap is saved. Recent ratings shape the next prompt, so the edition drifts toward what you keep marking relevant.

~10 min

Run your own

  1. Create a Telegram bot

    Message @BotFather, send /newbot, and copy the token.

  2. Get an Anthropic API key

    From console.anthropic.com — a daily brief costs a few cents.

  3. Fill in .env

    Paste both secrets; pick your delivery time and timezone.

  4. Find your chat ID

    Start the bot, message it, send /chatid, and paste the number back into .env.

  5. Wake up informed

    Restart it — or send /brief to get one right now.

The whole thing

# install & configure
$ pip install -r requirements.txt
$ cp .env.example .env # add your 2 keys

# run it
$ python bot.py
